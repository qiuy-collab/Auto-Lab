# Example: Big Data Processing Report

## Structure

```
big-data-processing-report/
├── 需求/
│   └── 《大数据处理技术》.docx    # 老师发的作业要求
├── 交付/
│   ├── 大数据处理技术课程大作业报告.docx  # 填好的报告
│   ├── code/                       # Python 脚本
│   ├── data/                       # 原始数据 + 清洗后数据
│   └── output/                     # 分析结果（CSV + 图表）
└── 效果图/                         # 报告预览截图（待添加）
```

## Output Highlights

- 12 张 AI 生成的终端/IDE 截图（Ubuntu + Hadoop + Hive + Spark）
- 完整的代码交付（数据生成、清洗、分析、可视化）
- 图表输出（柱状图、趋势图、饼图）
- 学生口吻的报告内容
