Auto Lab

AI 驱动的大学实验报告自动填写工具。给 AI agent 一份需求文档 + 一个 Word 模板，它帮你把报告写好、截图配好、图表插好、格式调好，直接交。

GitHub 功能特性

功能特性

三条截图路线

ai_simulated: AI 生成逼真的终端、IDE、配置截图

browser_capture: 本地浏览器截图（自己的前端页面）

diagram_assets: 流程图、ER 图、数据流图自动生成

模板安全填充

保留模板结构，清理占位符和格式指令，用学生口吻重写内容

预任务支持

自动检测是否需要先完成代码、数据等预任务，再写报告

40+ 验收检查

内置模板完整性、目录、正文质量、图例、交付包等全面检查

一键打包交付

自动生成 submit/ 文件夹 + submit.zip，代码/数据/报告完整交付

Agent 友好

标准化工作流，Agent Skills Standard 兼容，支持多种 runtime

工作流程

需求文档 + Word 模板
       ↓
  自动安装依赖 + 环境检查
       ↓
  分析需求 → 规划截图和图表路线
       ↓
  🔴 确认分析结果（agent 停下来等你）
       ↓
  生成 AI 截图 / 浏览器截图 / 流程图
       ↓
  填写模板 → 清理占位符 → 插入图片
       ↓
  🔴 检查输出文档（agent 停下来等你）
       ↓
  打包交付（submit/ 文件夹 + submit.zip）
